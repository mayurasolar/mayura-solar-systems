# Image Assets for Mayura Solar Systems Website

## Required Images

Please add the following images to this directory:

### 1. Logo (REQUIRED)
- **Filename**: `logo.png`
- **Size**: 200x200px recommended
- **Format**: PNG with transparent background
- **Usage**: Navbar, Footer, Hero section

### 2. Hero/Banner Images (Optional but Recommended)
- **Filename**: `hero-banner.jpg`
- **Size**: 1920x1080px
- **Format**: JPG or PNG
- **Usage**: Home page hero section background

### 3. Product Images (Recommended)
Add your actual product photos:

- `etc-water-heater.jpg` - ETC Solar Water Heater
- `street-light.jpg` - Solar Street Light
- `solar-panel.jpg` - Solar Panel
- `etc-tubes.jpg` - ETC Replacement Tubes
- `garden-light.jpg` - Solar Garden Lights
- `water-pump.jpg` - Solar Water Pump

**Recommended specs**:
- Size: 800x600px
- Format: JPG or PNG
- Optimized for web (compressed)

## How to Add Images

1. Copy your logo file to this folder as `logo.png`
2. Copy product images with the filenames listed above
3. Restart the development server if running
4. Images will automatically appear on the website

## Image Optimization

Next.js automatically optimizes images for:
- Faster loading
- Responsive sizing
- Modern formats (WebP)

No additional configuration needed!

## Temporary Placeholders

The website currently uses:
- Emoji icons as temporary placeholders
- Gradient backgrounds
- The site will work without images, but adding them will make it look professional
